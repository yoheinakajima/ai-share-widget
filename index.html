AI Share Widget

A tiny, zero-dependency widget that renders a row of buttons—one per AI service—and opens your prompt in the chosen tool (ChatGPT, Claude, Perplexity, Gemini). Recommended import via jsDelivr. Source on GitHub.

Minimal Mobile-friendly Drop-in Extensible

Generate use cases with AI

Example

This demonstrates the AI Share widget: clicking a service sends a prompt that includes this page’s content as context.

Quick Start (jsDelivr)

<!-- 1) Import the widget (use @main now; switch to @v1 after tagging a release) -->
<script src="https://cdn.jsdelivr.net/gh/yoheinakajima/ai-share-widget@main/ai-share.js"></script>

<!-- 2) Add a container where the row of buttons should appear -->
<div id="ai-buttons"></div>

<!-- 3) Mount it with your prompt (string or function) -->
<script>
  AiShare.mountButtons('#ai-buttons', () => {
    const text = document.body.innerText.replace(/\s+/g,' ').trim().slice(0, 1800);
    return [
      "Brainstorm 12 distinct, concrete ways to integrate this 'Ask AI' widget into different websites and apps.",
      "Vary by product type (docs, blogs, ecommerce, dashboards, SaaS), user intent, and placement.",
      "For each idea: provide a short title and one sentence rationale.",
      "Use the following site content as context when relevant:",
      text
    ].join("\\n");
  }, { services: ['chatgpt','claude','gemini','perplexity'] });
</script>

API

AiShare.mountButtons(container, promptOrFn, { services?: string[], size?: 'md'|'sm' })
AiShare.openDirect(serviceKey, prompt)
AiShare.registerService(key, { label, url, param })
AiShare.registry.services

Customize services

Add or override destinations if URLs/params change:

AiShare.registerService('poe', { label:'Poe', url:'https://poe.com/search', param:'q' });

What it’s good for

Let users take anything they see—docs, help pages, product lists, dashboards—and continue in their preferred AI: summarize, ideate, compare, translate, draft, or propose actions.

This page doubles as context for the demo prompt above.
