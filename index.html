AI Share Widget

A tiny script that opens your prompt in the AI tool of your choice (ChatGPT, Claude, Perplexity, Gemini). One import line. Drop a dropdown + Send button anywhere on your page. Open source: github.com/yoheinakajima/ai-share-widget

Tip: limit services by key: { services: ['chatgpt','claude'] }

Use it via CDN (one line import): 
<script src="https://yoheinakajima.github.io/ai-share-widget/ai-share.js"></script>
<script>
  // Mount a dropdown anywhere:
  AiShare.mountDropdown('#some-container', () => "Summarize " + document.title + " (" + location.href + ")");
  // Or open a specific model directly:
  // AiShare.openDirect('chatgpt', "Your prompt here");
</script>
Alternative via jsDelivr (versioned): 
<script src="https://cdn.jsdelivr.net/gh/yoheinakajima/ai-share-widget@v1/ai-share.js"></script>
