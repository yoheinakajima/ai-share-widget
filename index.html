AI Share Widget

A tiny script that opens your prompt in ChatGPT, Claude, Perplexity, or Gemini. One import line. Open source: github.com/yoheinakajima/ai-share-widget

Pick a prompt about this page:
Use it via CDN (one line import): 
<script src="https://yoheinakajima.github.io/ai-share-widget/ai-share.js"></script>
<script>
  // Attach to your Ask AI button; dropdown appears on click; selection opens immediately:
  AiShare.attachLauncher('#ask', () => "Summarize " + document.title + " (" + location.href + ")");
</script>
Alternative via jsDelivr (versioned): 
<script src="https://cdn.jsdelivr.net/gh/yoheinakajima/ai-share-widget@v1/ai-share.js"></script>
